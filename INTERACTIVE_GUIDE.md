# 🎨 Interactive Marketing Image Generator

## 🆕 완전히 새로운 인터랙티브 GUI!

**마우스로 드래그해서 텍스트 위치를 자유롭게 조정**할 수 있습니다!

### ✨ 주요 기능

✅ **스크린샷 추가** - 1개 또는 여러 개
✅ **배경 이미지 선택** - 원하는 배경 사용
✅ **텍스트 자유 배치** - 마우스 드래그로 위치 조정
✅ **실시간 미리보기** - 변경사항 즉시 확인
✅ **완전한 커스터마이징** - 폰트, 크기, 색상

## 🚀 빠른 시작

```bash
python gui_interactive.py
```

## 📖 사용 방법

### 1단계: 이미지 준비

**옵션 A: 스크린샷 추가**
- "📄 스크린샷 추가" 버튼 클릭
- iPhone에서 캡처한 이미지 선택
- 여러 개 선택 가능 (자동으로 배치)

**옵션 B: 배경 이미지 사용**
- "🖼️ 배경 이미지 선택" 버튼 클릭
- 원하는 배경 이미지 선택
- 그 위에 스크린샷이 추가됨

### 2단계: 텍스트 추가

1. **텍스트 입력 칸**에 원하는 문구 입력
   - 예: "Treat yourself"
   
2. **폰트 크기 조정**
   - 슬라이더로 30~150 사이 선택
   - 기본값: 90px

3. **폰트 선택**
   - Helvetica (기본)
   - SF Pro (iOS 스타일)
   - Roboto (Android)
   - Montserrat (우아함)

4. **텍스트 색상 선택**
   - "🎨 텍스트 색상" 버튼 클릭
   - 원하는 색상 선택

5. **"➕ 텍스트 추가" 버튼 클릭**

### 3단계: 텍스트 위치 조정 (핵심 기능!)

**마우스로 드래그하여 텍스트를 원하는 위치로 이동:**

1. 캔버스에서 **텍스트를 클릭**
2. **마우스를 누른 채로 드래그**
3. 원하는 위치에서 **마우스 버튼 릴리스**

💡 **Tip**: 여러 텍스트를 추가하고 각각 다른 위치에 배치할 수 있습니다!

### 4단계: 저장

- "💾 이미지 저장" 버튼 클릭
- 파일 이름 입력
- 저장!

## 🎨 디자인 팁

### 텍스트 배치 가이드

**상단 텍스트 (제목)**
- Y 위치: 150-300
- 크기: 80-120px
- 용도: 주요 메시지

**중단 텍스트**
- Y 위치: 1200-1500
- 크기: 60-80px
- 용도: 보조 설명

**하단 텍스트**
- Y 위치: 2400-2600
- 크기: 40-60px
- 용도: 추가 정보, 면책 조항

### 색상 조합

**밝은 배경:**
- 제목: RGB(60, 120, 255) - 파란색
- 부제목: RGB(80, 80, 80) - 회색

**어두운 배경:**
- 제목: RGB(255, 255, 255) - 흰색
- 부제목: RGB(200, 200, 200) - 연한 회색

**그라디언트 배경:**
- 대비가 높은 색상 사용
- 밝은 부분에 어두운 텍스트
- 어두운 부분에 밝은 텍스트

4. 저장

### 3. 시즌별 프로모션

1. 기본 레이아웃 저장

## ❓ FAQ

**Q: 텍스트가 드래그되지 않아요**
A: 텍스트를 정확히 클릭했는지 확인하세요. 텍스트 주변 ±300px 범위 내 클릭

**Q: 텍스트가 너무 작아요/커요**
A: 폰트 크기 슬라이더로 30~150 사이 조정

**Q: 여러 텍스트를 추가할 수 있나요?**
A: 네! 원하는 만큼 추가 가능. 각각 개별적으로 드래그 가능

**Q: 텍스트를 수정하려면?**
A: 목록에서 삭제 후 새로 추가

**Q: 배경 없이 스크린샷만?**
A: 스크린샷만 추가하면 자동으로 그라디언트 배경 생성

**Q: 저장된 이미지 해상도는?**
A: 정확히 1290x2796 픽셀 (App Store 규격)
